### 環境

- iPad
- safari

### 操作

second-componentのページを開き、以下のいずれかのボタンを押す

- "go back with history.back()"ボタン
- "go back by router"ボタン
- ブラウザの"戻る"ボタン


### 期待する動作

- いずれのボタンでも確認ダイアログが表示され、OKを押すと画面遷移する

### 実際の動作

- "go back with history.back()"ボタン -> 機能する
- "go back by router"ボタン -> 機能する
- ブラウザの"戻る"ボタン -> 機能しない
